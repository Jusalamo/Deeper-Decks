Deeper: The Connection Card Game

Deeper is a single-file, responsive web application designed to spark meaningful conversations and fun interactions, whether you are on a date night or having a casual get-together with friends.

This project is built using pure HTML, CSS (via Tailwind CSS CDN), and vanilla JavaScript.

Choose a Deck: Select one of the three card decks from the control panel:

Romantic: For couples focusing on deep love, future plans, and intimacy.

Platonic: For friends or groups, focusing on personal growth, shared experiences, and fun memories.

Naughty: A spicier deck for partners looking for provocative questions and dares.

Shuffle: The deck is automatically shuffled when selected. You can re-shuffle at any time using the "Shuffle Current Deck" button.

Draw: Click "Draw Next Card" to reveal a question or a dare.

Connect: Answer the question honestly or complete the dare!
